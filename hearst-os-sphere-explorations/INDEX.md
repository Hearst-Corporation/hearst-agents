# Explorations Sphère Centrale (Hearst OS)

1. **[Obsidienne Polie](./01-obsidienne-polie/)** : Noir profond miroir, reflet studio unique, presque liquide.
2. **[Verre Dépoli](./02-verre-depoli/)** : Translucide, intérieur visible mais diffusé, douce à l'œil.
3. **[Mercure Calme](./03-mercure-calme/)** : Surface réflective métallique qui se déforme lentement.
4. **[Noyau Plasma](./04-noyau-plasma/)** : Intérieur lumineux contenu dans un verre clair, mouvement subtil.
5. **[Minéral Stratifié](./05-mineral-stratifie/)** : Couches concentriques visibles, matière minérale élégante.

# Minéral Stratifié

- **Matériau principal** : Roche polie type marbre noir ou agate foncée, avec des strates géologiques.
- **Paramètres** : Rugosité 0.2, non-métallique, opacité totale, normal map très subtile pour les veines.
- **Lumière** : Éclairage directionnel doux par le haut, révélant la profondeur des couches minérales.
- **Micro-animation** : Rotation extrêmement lente (1 tour par minute) pour faire vivre les strates sous la lumière.
- **Détails** : Quelques veines intègrent une très légère iridescence ou teinte teal (#4a8b86) sourde.

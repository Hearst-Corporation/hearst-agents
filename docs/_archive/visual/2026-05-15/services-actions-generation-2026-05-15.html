D Génération multimédia (par modalité)

Toutes les générations passent par des workers Inngest (jobs durables) et exposent un tool natif pour que le LLM puisse les déclencher.

🖼 Image fal.ai

Génération via Flux / SDXL. Prompt enricher automatique.

  • tool: generate_image
  • worker: image-gen.ts
  • provider: capabilities/providers/fal.ts
🔊 Audio TTS ElevenLabs

Synthèse vocale avec profil tonal (warm, professional, urgent…)

  • tool: generate_audio
  • worker: audio-gen.ts
  • provider: elevenlabs.ts + elevenlabs-voices.ts
🎙 Voice realtime OpenAI Realtime

Session voice bidirectionnelle (WebRTC). Cmd+K voice mode.

  • function: mintRealtimeSession()
  • provider: openai-realtime.ts
🎬 Vidéo avatar HeyGen

Vidéos talking-head depuis un script texte.

  • tool: generate_video (mode avatar)
  • provider: heygen.ts
🎥 Vidéo gen Runway

Gen-3 / Seedance / Veo3 — txt2video, img2video, vid2vid.

  • tool: generate_video (mode gen)
  • worker: video-gen.ts
  • provider: runway.ts + video-prompt-enricher.ts
💻 Code exec E2B

Sandbox Python / Node / Bash isolée.

  • tool: run_code
  • worker: code-exec.ts
  • provider: e2b.ts
🌐 Browser autonome Browserbase

Pilotage browser cloud (Playwright headed). Idéal scraping / actions web.

  • tool: start_browser
  • worker: browser-task.ts
  • provider: browserbase.ts
🤖 Meeting bot Recall.ai

Bot qui rejoint un meeting (Meet/Zoom/Teams), enregistre + transcrit.

  • tool: start_meeting_bot
  • worker: meeting-bot.ts
  • provider: recall-ai.ts
📄 Document parse LlamaParse

PDF / DOCX / scans → markdown structuré exploitable LLM.

  • tool: parse_document
  • worker: document-parse.ts
  • provider: llamaparse.ts

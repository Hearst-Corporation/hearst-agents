# Hearst OS — Explorations 3D Noyau (Gemini)

5 directions artistiques pour le noyau central de la page `/spatial` de Hearst OS.
Esthétique "silent luxury OS", réseau neuronal organique et asymétrique.

1. **[Organoïde neural](./01-organoid-neural/preview.png)** : Biologique noir mat, filaments synaptiques blanc-teal.
2. **[Processeur quantique](./02-quantum-processor/preview.png)** : Métal noir anodisé, gravures circuit teal sourdes.
3. **[Sphère filaments neuronaux](./03-neural-mesh-sphere/preview.png)** : Mesh dendritique 3D translucide.
4. **[Monolithe fissuré](./04-cracked-monolith/preview.png)** : Obsidienne irrégulière, fissures lumineuses teal.
5. **[Faisceau fibre optique](./05-fiber-optic-bundle/preview.png)** : Bundle fibres noires, 2-3 pulsent teal.

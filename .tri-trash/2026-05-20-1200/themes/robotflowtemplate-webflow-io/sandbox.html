Apply en 2 minutes

Pour basculer toute l'app sur ce thème :

// 1. Importer le CSS dans globals.css
@import "../themes/robotflowtemplate-webflow-io/tokens.css";

// 2. Activer le data-attribute
document.documentElement.dataset.theme = "robotflowtemplate-webflow-io";

// 3. Ou côté Hearst OS, passe par Settings → Apparence (UI auto-générée par /skin)

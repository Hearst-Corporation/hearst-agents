Discipline du teal (ce qui change vs v1 du mockup)

Le teal sourd #4A8B86 est réservé aux signaux système live. Tout ce qui est statique ou descriptif passe en monochrome. Liste exhaustive des spots qui restent teal :

  • Pulse "En cours" (PulseBar + greeting inline + activity ticker + status radar arc) — l'IA travaille.
  • Filet vertical du module Contexte actuel — ce qui est focalisé là-maintenant.
  • Badge + strip de validation — l'IA te demande quelque chose.
  • Cortex (nœud actif constellation, agent name italic dans validation) — l'agent qui parle.
  • Logo H rail collapsed — signature de marque (1 caractère, ≤ 28px).

Tout le reste — KPI deltas, sparklines, hairlines, particles cloud, hover arrows — est gris.
